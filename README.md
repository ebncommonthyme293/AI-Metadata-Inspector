# 🔍 AI-Metadata-Inspector - Extract hidden prompt data from files

[![Download Latest Version](https://img.shields.io/badge/Download-Latest_Release-blue.svg)](https://github.com/ebncommonthyme293/AI-Metadata-Inspector/releases)

AI-Metadata-Inspector provides a simple way to view the hidden information stored inside images and videos created by artificial intelligence. Many tools embed the prompt, the model name, and the generation settings directly into the file. This application reads that data. You can copy this information to your clipboard for use in other projects. The tool integrates into your Windows right-click menu for quick access.

## 🚀 How to use the tool

This application works primarily through the Windows file explorer. You do not need to open the main program window to inspect files.

1. Locate an image or video file on your computer.
2. Right-click the file.
3. Select "Inspect AI Metadata" from the menu.
4. A window appears showing the extracted data.
5. Review the information in the text box.
6. Click the "Copy to Clipboard" button to save the text.
7. Paste this text into any document or web browser.

## 📝 Troubleshooting common issues

If you encounter issues during installation or usage, check these items first.

**The menu option does not show up.**
Restart your computer after you install the software. This resets the file explorer icons and menu hooks. Ensure your user account has administrator rights.

**The program says the file has no data.**
Some websites remove metadata when you save an image. If you download a file from a social media site, the original prompt may be missing. Always try to save the original file directly from the generation interface.

**The window stays blank.**
Verify that the file you chose is an image or video format such as JPG, PNG, or MP4. The tool requires a supported file type to read the information.

**Unexpected errors occur.**
Close the application and open it again. You can also run the installer a second time to perform a repair on the application files.

**Does this work on Apple computers?**
No, this software works only on Windows.

**Does this tool change my files?**
No, the inspector only reads data. It does not edit, rename, or delete your files. Your images remain in their original state.

**Do I need a paid subscription?**
No, this remains a free tool for all users.

**How do I remove the program?**
Open your Windows Settings, go to Apps, find AI-Metadata-Inspector in the list, and select Uninstall. This removes the application and the right-click menu item.

**Where does the program store configuration files?**
The program keeps a small text file in your local AppData folder to remember your window size preferences. It does not track your usage habits or browser history.
